# MICHDEV BUILD SYSTEM 2025

**Gulp 5 + Esbuild + TypeScript + Nunjucks + Sass + WebP + PHP**  


---

## Особенности

- Компиляция **SCSS** с автопрефиксом и минификацией
- Сборка **TypeScript** через **Esbuild** с минификацией и sourcemap
- Рендеринг **HTML** через **Nunjucks** с JSON-данными (глобальные, страницы, компоненты)
- Оптимизация изображений и конвертация в **WebP**
- Копирование шрифтов, медиа и PHP файлов
- Локальный сервер с **BrowserSync** и автоматическим обновлением
- Watch всех исходных файлов
